实现增强版“魔术橡皮擦”功能，集成大小选择、可视化光标、撤销/重做及动态容差调整。

## 1. UI 变更
- **右侧工具栏升级**：
  - **橡皮擦菜单**：点击展开，提供：
    - **大小选择**：3 个预设大小（小 10px / 中 30px / 大 60px）。
    - **容差滑块**：允许用户实时调整颜色匹配的灵敏度（范围 0-100）。
  - **撤销/重做按钮**：新增两个独立按钮，分别用于回退和重做操作。

- **可视化反馈**：
  - **动态光标**：在图片上移动鼠标时，显示一个跟随的圆形虚线框，其实时大小对应当前选择的橡皮擦尺寸。

## 2. 核心逻辑实现
- **历史记录系统**：
  - 实现双栈结构：`_undoStack` 和 `_redoStack`。
  - 每次擦除操作前，将当前状态推入 `undoStack`，清空 `redoStack`。
  - 点击撤销/重做时，在两个栈之间移动状态并更新显示。

- **魔术擦除算法 (`_performMagicErase`)**：
  - **输入**：点击坐标、当前半径、当前容差值。
  - **处理**：
    1. 锁定图片像素数据。
    2. 获取点击点颜色为“基准色”。
    3. 遍历圆形区域内的所有像素。
    4. 计算每个像素与基准色的 **RGB 欧氏距离**。
    5. 若距离小于 `tolerance`，将 Alpha 通道置为 0（完全透明）。
  - **输出**：更新后的像素数据流。

## 3. 交互流程
- 用户点击工具栏橡皮擦 -> 选择大小/调整容差 -> 鼠标移入图片（显示圆圈） -> 点击目标区域 -> 区域内相近颜色被“抹除”为透明 -> 用户可点击撤销回退不满意的操作。
